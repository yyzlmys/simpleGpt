- 若要运行该 `FastAPI` 项目，需在 `/fastApi` 目录下增加 `.env` 文件，里面填写所需的 `Key` :
  ```env
    # openai key
    OPENAI_API_KEY=your_key
    OPENAI_API_BASE=your_base_url

    # 百度千帆
    QIANFAN_AK=your_key
    QIANFAN_SK=your_key

    # 讯飞星火
    IFLYTEK_SPARK_APP_ID=your_key
    IFLYTEK_SPARK_API_KEY=your_key
    IFLYTEK_SPARK_API_SECRET=your_key
    IFLYTEK_SPARK_API_URL=your_key
    IFLYTEK_SPARK_llm_DOMAIN=your_key

    # 清华智谱
    ZHIPUAI_API_KEY=your_key

    # 阿里通义千问
    DASHSCOPE_API_KEY=your_key

    # Google Serper
    SERPER_API_KEY=your_key
  ```
